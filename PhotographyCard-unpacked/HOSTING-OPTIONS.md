## 🎯 **Quick Deploy to GitHub Pages:**

```bash
# 1. Create repo on GitHub (nitin-art-studio)

# 2. Push your code
cd ~/Downloads/PhotographyCard
git branch -M main
git add .
git commit -m "NITIN ART STUDIO business card"
git remote add origin https://github.com/YOUR-USERNAME/nitin-art-studio.git
git push -u origin main

# 3. Enable GitHub Pages in Settings → Pages
# Source: GitHub Actions

# 4. Wait 2-3 minutes

# 5. Visit: https://YOUR-USERNAME.github.io/nitin-art-studio/
```

**Done! Your site is LIVE! 🎉**
