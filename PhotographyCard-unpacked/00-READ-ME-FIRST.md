### 1. Add Images ⚠️ REQUIRED

```bash

# - Phone number  
# - Email
# - Social media links
```

### 3. Run It!
```bash

cd ~/Downloads/PhotographyCard
./run.sh

# Then open: http://localhost:5000
```

## 🆘 Need Help?

**Images not showing?**
- Check: `wwwroot/images/logo.png` exists
- Check: `wwwroot/images/qr-code.png` exists

**App won't run?**
```bash
dotnet --version  # Should be 8.0+
dotnet restore
dotnet build
```

**Links not working?**
- Phone must have: `+1-647-...`
- WhatsApp must have: `1647...` (no + or -)
- URLs must start with: `https://`
